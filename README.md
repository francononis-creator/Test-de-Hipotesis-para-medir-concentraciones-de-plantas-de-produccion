# Test de Hipotesis para medir concentraciones
En el siguiente proyecto se realizo un test de hipotesis para cada planta de produccion de una empresa farmaceuitca con el fin de saber si la concentracion de los productos respeta la normativa necesaria, ademas de un analisis descriptivo.
In the following project, a hypothesis test was conducted for each production plant of a pharmaceutical company to determine whether the product concentrations comply with the required standards, along with a descriptive analysis.
